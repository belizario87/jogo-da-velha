# Jogo da velha 

## Descrição
Jogo da velha para 2 jogadores ou contra IA desenvolvido em Javascript.

## O que foi usado
- Html
- Css
- Javascript

## Conceitos
* Foram usados conceitos de logica de programação
* Manipulação de elementos DOM através de javascript
* Recursividade
* Elementos graficos
